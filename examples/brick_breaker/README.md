# Brick Breaker - Example

This is a simple brick breaker game. Move the paddle around so the ball does not reach the bottom. Destroy as many bricks as you can to procede to the next leve.

## Controls

| | |
|--|--|
| `←` | move paddle left |
| `→` | move paddle right |
| `*any key*` | to start |
